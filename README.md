GoPOS recruitment Task

CRUD application connected to GoPOS API.

## [Live demo](https://recruitment-task-rho.vercel.app/)

Built using:

- React
- Typescript
- NextJS
- Axios & React Query
- Bootstrap

To run the app type:

```bash
npm install
# or
yarn
```

Then run the development server:

```bash
npm run dev
# or
yarn dev
```

Enjoy!
